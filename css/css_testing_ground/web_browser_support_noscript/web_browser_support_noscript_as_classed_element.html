Results

Web browser support for the HTML noscript element as a classed element: IE 6+, ED12*, FF1.5+, SF3.1+, CH5+**, OP15+.

Web browser no support for the HTML noscript element as a classed element: OP12.17-.

* Apparently it is not possible to disable JavaScript on ED13. Therefore, ED13 not tested.
** Apparently it is not possible to disable JavaScript on CH4-. Therefore, CH4- not tested.

* Apparently cannot disable JavaScript on ED13 and CH4-. Therefore, did not test ED13 and CH4-.
