Technologies And APIs Not Supported In Microsoft Edge
https://blog.learnwebcoding.com/?p=8

Technologies and APIs not supported in Microsoft Edge (These technologies will continue to be supported in Internet Explorer on Windows 10 for users and enterprises who depend on them.): ActiveX, Browser Helper Objects (BHO), document modes, Vector Markup Language (VML), VBScript, attachEvent/removeEvent, currentStyle, conditional comments, IE8 layout quirks, DirectX filters and transitions.
